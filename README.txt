1) Go to google chrome and type in chrome://extensions
2) Check in Developer mode in the top right corner
3) Click on Load unpacked extension.
4) Load in the Summarize folder

Format----------
Summarizer(Folder name)
	lib(Folder name)
		content.js
		jquery-2.2.1.js
	icon.png
	manifest.json
	popup.html
	popup.js
